# projects

Larger, multi-session practice builds — bigger than a `playgrounds/` scratch, smaller than a product. Good source: the "Projects" tab on each roadmap.sh roadmap.

Each project = its own folder with a README from [`../templates/project.md`](../templates/project.md):

```
projects/<name>/
├── README.md     # goal, stack, milestones, log
└── ...code...
```

Tag the related roadmaps in the README frontmatter (`roadmaps: [backend, docker]`) so a project counts toward what you're learning.
