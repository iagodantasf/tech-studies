# Elasticsearch — resources

> roadmap.sh: https://roadmap.sh/elasticsearch

## Books
- **Elasticsearch: The Definitive Guide** (Clinton Gormley & Zachary Tong) — the classic conceptual intro to search, analysis, and distribution.
- **Elasticsearch in Action** (Madhusudhan Konda) — hands-on, modern coverage of the Query DSL and aggregations.
- **Relevant Search** (Doug Turnbull & John Berryman) — how to actually tune relevance and scoring in Lucene-based engines.
- **Mastering Elasticsearch** (Rafal Kuc & Marek Rogozinski) — deeper dive into internals, scaling, and cluster administration.

## Courses / practice
- [Elasticsearch official documentation](https://www.elastic.co/docs) — canonical reference for APIs, mappings, and the Query DSL.
- [Elastic Search Labs & tutorials](https://www.elastic.co/search-labs) — official guided tutorials including vector / semantic search.
- [Quickstart guide](https://www.elastic.co/docs/solutions/search/get-started) — official getting-started path for indexing and searching your first data.
- [Query DSL reference](https://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl.html) — searchable docs for every query type with examples.
