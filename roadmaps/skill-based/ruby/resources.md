# Ruby — resources

> roadmap.sh: https://roadmap.sh/ruby

## Books
- **The Well-Grounded Rubyist** (David A. Black) — the best deep, modern introduction to how Ruby actually works.
- **Programming Ruby (The Pickaxe)** (Dave Thomas et al.) — the classic comprehensive reference for the language and stdlib.
- **Practical Object-Oriented Design in Ruby (POODR)** (Sandi Metz) — essential OO design wisdom taught through Ruby.
- **Eloquent Ruby** (Russ Olsen) — teaches idiomatic, expressive Ruby style and conventions.

## Courses / practice
- [Official Ruby docs](https://www.ruby-lang.org/en/documentation/) — canonical language site with guides and references.
- [Ruby core/stdlib API docs](https://docs.ruby-lang.org/en/) — authoritative reference for every built-in class and method.
- [The Odin Project — Ruby](https://www.theodinproject.com/paths/full-stack-ruby-on-rails) — free, project-based full curriculum.
- [Exercism Ruby track](https://exercism.org/tracks/ruby) — hands-on exercises with mentoring to build fluency.
