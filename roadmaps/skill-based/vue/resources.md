# Vue — resources

> roadmap.sh: https://roadmap.sh/vue

## Books
- **Fullstack Vue** (Hassan Djirdeh et al.) — project-driven guide to components, Vuex/Pinia, and routing.
- **Vue.js 3 Cookbook** (Heitor Ramon Ribeiro) — recipe-style coverage of the Composition API and real-world patterns.
- **The Vue Handbook** (Flavio Copes) — concise, free reference that's great as a quick refresher.

## Courses / practice
- [Vue.js docs (vuejs.org)](https://vuejs.org/guide/introduction.html) — the official guide and API reference; the canonical starting point.
- [Vue School](https://vueschool.io/) — official-partner video courses spanning beginner to advanced Vue 3.
- [Vue Mastery](https://www.vuemastery.com/) — high-quality tutorials from the team behind much of the Vue ecosystem.
- [Nuxt docs](https://nuxt.com/docs) — official guide for building full-stack/SSR Vue apps with Nuxt.
