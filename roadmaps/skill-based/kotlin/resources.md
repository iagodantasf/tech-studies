# Kotlin — resources

> roadmap.sh: https://roadmap.sh/kotlin

## Books
- **Kotlin in Action** (Jemerov, Isakova) — written by the JetBrains team; the definitive language book.
- **Atomic Kotlin** (Eckel, Isakova) — incremental, exercise-driven path from basics to advanced features.
- **Effective Kotlin** (Marcin Moskała) — idiomatic best practices for writing clean, safe Kotlin.
- **Kotlin Coroutines** (Marcin Moskała) — focused, in-depth guide to coroutines and Flow.

## Courses / practice
- [Kotlin Documentation](https://kotlinlang.org/docs/home.html) — the official, comprehensive language reference.
- [Kotlin Koans](https://play.kotlinlang.org/koans/overview) — interactive exercises covering core language features.
- [JetBrains Academy — Kotlin](https://hyperskill.org/tracks?category=4) — guided, project-based Kotlin learning tracks.
- [Kotlin Playground](https://play.kotlinlang.org/) — run and experiment with Kotlin directly in the browser.
