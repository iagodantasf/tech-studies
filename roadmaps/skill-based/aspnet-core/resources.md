# ASP.NET Core — resources

> roadmap.sh: https://roadmap.sh/aspnet-core

## Books
- **Pro ASP.NET Core** (Adam Freeman) — the comprehensive, regularly-updated reference for the framework.
- **ASP.NET Core in Action** (Andrew Lock) — clear, in-depth coverage from fundamentals to advanced topics.
- **C# in Depth** (Jon Skeet) — essential for mastering the language underneath ASP.NET Core.
- **Entity Framework Core in Action** (Jon P. Smith) — the go-to book for EF Core data access.

## Courses / practice
- [ASP.NET Core Documentation](https://learn.microsoft.com/en-us/aspnet/core/) — the official Microsoft Learn docs.
- [Microsoft Learn — .NET](https://learn.microsoft.com/en-us/training/dotnet/) — free, hands-on guided learning paths.
- [.NET Microservices eShopOnContainers](https://github.com/dotnet-architecture/eShopOnContainers) — reference architecture and sample app.
- [Andrew Lock's .NET Blog](https://andrewlock.net/) — deep, reliable articles on ASP.NET Core internals.
