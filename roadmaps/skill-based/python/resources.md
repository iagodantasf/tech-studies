# Python — resources

> roadmap.sh: https://roadmap.sh/python

## Books
- **Fluent Python** (Luciano Ramalho) — the deep dive on idiomatic, Pythonic code and the data model.
- **Python Crash Course** (Eric Matthes) — fast, project-driven path from zero to building real apps.
- **Effective Python** (Brett Slatkin) — 90 concise best-practices that level up everyday code.
- **Python Cookbook** (Beazley & Jones) — recipe-style solutions for real-world problems.

## Courses / practice
- [Official Python docs & tutorial](https://docs.python.org/3/tutorial/) — the canonical, always-current reference.
- [Real Python](https://realpython.com/) — high-quality tutorials across the whole ecosystem.
- [Automate the Boring Stuff](https://automatetheboringstuff.com/) — free book/course for practical scripting.
- [Exercism Python track](https://exercism.org/tracks/python) — exercises with mentor feedback.
