# Kubernetes — resources

> roadmap.sh: https://roadmap.sh/kubernetes

## Books
- **Kubernetes Up & Running** (Burns, Beda, Hightower & Evenson) — the canonical from-zero guide, co-authored by Kubernetes founders.
- **The Kubernetes Book** (Nigel Poulton) — concise, frequently-updated, beginner-friendly tour of the core objects.
- **Kubernetes in Action** (Marko Lukša) — deep, example-driven coverage of how the pieces fit together.
- **Programming Kubernetes** (Hausenblas & Schimanski) — for writing controllers, operators, and CRDs against the API.

## Courses / practice
- [Kubernetes official documentation](https://kubernetes.io/docs/home/) — authoritative reference and concept guides.
- [Kubernetes Basics interactive tutorial](https://kubernetes.io/docs/tutorials/kubernetes-basics/) — official browser-based walkthrough.
- [Killercoda Kubernetes scenarios](https://killercoda.com/kubernetes) — free interactive hands-on labs in a real cluster.
- [Helm documentation](https://helm.sh/docs/) — official docs for the de facto Kubernetes package manager.
