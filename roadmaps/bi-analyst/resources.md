# BI Analyst — resources

> roadmap.sh: https://roadmap.sh/bi-analyst

## Books
- **Storytelling with Data** (Cole Nussbaumer Knaflic) — the canonical guide to communicating insights with clear, honest charts.
- **The Big Book of Dashboards** (Wexler, Shaffer, Cotgreave) — real-world dashboard patterns for common business scenarios.
- **SQL for Data Analysis** (Cathy Tanimura) — practical analytical SQL (window functions, cohorts, funnels) aimed at analysts.
- **Information Dashboard Design** (Stephen Few) — foundational principles for at-a-glance monitoring displays.

## Courses / practice
- [Microsoft Power BI documentation](https://learn.microsoft.com/en-us/power-bi/) — official learning paths and DAX/modeling reference.
- [Tableau Learning](https://www.tableau.com/learn/training) — free official training videos and how-to guides.
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/) — hands-on analytical SQL tutorial from basics to advanced.
- [Kaggle: Data Visualization](https://www.kaggle.com/learn/data-visualization) — short, practical visualization micro-course.
