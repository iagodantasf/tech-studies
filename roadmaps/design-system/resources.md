# Design System — resources

> roadmap.sh: https://roadmap.sh/design-system

## Books
- **Design Systems** (Alla Kholmatova, Smashing Magazine) — the canonical book on building and maintaining design systems.
- **Atomic Design** (Brad Frost) — the foundational methodology for composing UI from atoms to pages.
- **Expressive Design Systems** (Yesenia Perez-Cruz) — pragmatic guidance on flexibility, contribution, and governance.

## Courses / practice
- [Design Systems Repo](https://designsystemsrepo.com/) — curated gallery of real design systems, articles, and tools.
- [Storybook Docs](https://storybook.js.org/docs) — the standard tool for building and documenting components in isolation.
- [Style Dictionary](https://styledictionary.com/) — official docs for transforming design tokens across platforms.
- [Material Design Guidelines](https://m3.material.io/) — a comprehensive reference system to study for structure and tokens.
