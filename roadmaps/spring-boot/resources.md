# Spring Boot — resources

> roadmap.sh: https://roadmap.sh/spring-boot

## Books
- **Spring in Action** (Craig Walls) — the standard hands-on introduction to Spring and Spring Boot.
- **Spring Boot Up & Running** (Mark Heckler) — practical, project-driven coverage of building Boot apps.
- **Spring Microservices in Action** (Carnell, Sánchez) — applying Spring Boot/Cloud to microservice architectures.
- **Pro Spring 6** (Cosmina et al.) — comprehensive reference for the Spring Framework internals.

## Courses / practice
- [Spring Boot Reference Documentation](https://docs.spring.io/spring-boot/index.html) — the authoritative Spring Boot docs.
- [Spring Guides](https://spring.io/guides) — short, focused official getting-started tutorials.
- [Spring Academy](https://spring.academy/) — VMware's official learning platform with free courses.
- [Baeldung Spring Tutorials](https://www.baeldung.com/spring-tutorial) — deep, practical articles on every Spring topic.
