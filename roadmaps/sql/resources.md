# SQL — resources

> roadmap.sh: https://roadmap.sh/sql

## Books
- **SQL in 10 Minutes, Sams Teach Yourself** (Ben Forta) — fast, practical intro to core SQL syntax across dialects.
- **Learning SQL** (Alan Beaulieu) — solid foundation in querying, joins, subqueries, and set operations.
- **SQL Performance Explained** (Markus Winand) — vendor-neutral guide to indexing and how the optimizer really uses them.
- **SQL Antipatterns** (Bill Karwin) — common schema and query mistakes and the patterns that fix them.

## Courses / practice
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/) — hands-on lessons from basics through advanced analytics in the browser.
- [SQLBolt](https://sqlbolt.com/) — interactive lessons with immediate query feedback, great for fundamentals.
- [PostgreSQL Tutorial](https://www.postgresqltutorial.com/) — clear, example-driven coverage of standard and advanced SQL.
- [SQLZoo](https://sqlzoo.net/) — exercise-first practice site covering joins, aggregates, and window functions. (LeetCode Database also has good problem sets.)
