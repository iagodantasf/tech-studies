# Claude Code — resources

> roadmap.sh: https://roadmap.sh/claude-code

## Books
- **Claude Code docs (official)** (Anthropic) — the canonical, always-current reference; treat it as the primary text.
- **Building Effective Agents** (Anthropic) — the engineering essay behind the agentic patterns Claude Code embodies.
- **The Pragmatic Programmer** (Hunt & Thomas) — the review and design judgment you need to drive an agent well.
- **Designing Data-Intensive Applications** (Martin Kleppmann) — systems grounding for the real projects you'll build with it.

## Courses / practice
- [Claude Code documentation](https://docs.anthropic.com/en/docs/claude-code/overview) — install, CLI, settings, hooks, MCP, subagents, and SDK references.
- [Model Context Protocol docs](https://modelcontextprotocol.io/) — the spec and SDKs for building MCP servers Claude Code can use.
- [Claude Agent SDK reference](https://docs.anthropic.com/en/api/agent-sdk/overview) — programmatic, headless agent building blocks.
- [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook) — runnable examples for tools, agents, and MCP integrations.
