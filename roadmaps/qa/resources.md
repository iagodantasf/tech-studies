# QA — resources

> roadmap.sh: https://roadmap.sh/qa

## Books
- **Foundations of Software Testing: ISTQB Certification** (Graham et al.) — the standard reference aligned to the ISTQB Foundation syllabus.
- **Lessons Learned in Software Testing** (Kaner, Bach & Pettichord) — context-driven wisdom from veteran testers; sharpens how you think about testing.
- **Agile Testing** (Crispin & Gregory) — how QA fits into agile teams and the whole-team approach to quality.
- **Explore It!** (Elisabeth Hendrickson) — a practical guide to exploratory testing that improves any tester's instincts.

## Courses / practice
- [Test Automation University](https://testautomationu.applitools.com/) — free, well-structured courses on Selenium, Cypress, Playwright, API testing and more.
- [Playwright docs](https://playwright.dev/) — official documentation and guides for modern end-to-end browser automation.
- [Postman Learning Center](https://learning.postman.com/) — official tutorials for API testing and collections.
- [Ministry of Testing](https://www.ministryoftesting.com/) — community, articles and courses spanning the whole QA discipline.
