# Cloudflare — resources

> roadmap.sh: https://roadmap.sh/cloudflare

## Books
- **Learning Cloudflare Workers** (community/O'Reilly-style guides) — practical introduction to building on the Workers edge runtime.
- **Web Performance in Action** (Jeremy Wagner) — background on caching, CDNs, and the optimizations Cloudflare automates.
- **Zero Trust Networks** (Evan Gilman & Doug Barth) — the conceptual foundation behind Cloudflare Access, Gateway, and Tunnel.

## Courses / practice
- [Cloudflare Docs](https://developers.cloudflare.com/) — the authoritative reference for every product, with quickstarts per service.
- [Cloudflare Workers Docs](https://developers.cloudflare.com/workers/) — tutorials and runtime APIs for the developer platform (Workers, KV, D1, R2).
- [Cloudflare Learning Center](https://www.cloudflare.com/learning/) — vendor-neutral explainers on DNS, CDN, DDoS, SSL/TLS, and Zero Trust.
- [Cloudflare Zero Trust Docs](https://developers.cloudflare.com/cloudflare-one/) — guided setup for Access, Gateway, Tunnel, and WARP.
