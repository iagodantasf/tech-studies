## Project ideas
- Write a complete PRD for a feature you wish existed in an app you use daily — include problem statement, success metrics, user stories, and a phased rollout plan.
- Run a prioritization exercise on a backlog of 15 hypothetical features using RICE and the Kano model, then defend your top 3 in a written one-pager.
- Build a product analytics dashboard from a public dataset, define a North Star metric and AARRR funnel, and propose two A/B tests to improve activation.
