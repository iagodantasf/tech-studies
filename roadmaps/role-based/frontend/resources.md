# Frontend — resources

> roadmap.sh: https://roadmap.sh/frontend

## Books
- **Eloquent JavaScript** (Marijn Haverbeke) — free, modern intro to JS and the browser; the canonical first read.
- **You Don't Know JS Yet** (Kyle Simpson) — deep dive into the language mechanics most tutorials skip.
- **Refactoring UI** (Adam Wathan & Steve Schoger) — practical visual design for developers who aren't designers.
- **Don't Make Me Think** (Steve Krug) — short, timeless guide to usability and web UX.

## Courses / practice
- [MDN Web Docs](https://developer.mozilla.org/) — the authoritative reference for HTML, CSS, JS, and web APIs.
- [The Odin Project](https://www.theodinproject.com/) — full free, project-based full-stack/frontend curriculum.
- [javascript.info](https://javascript.info/) — thorough, well-structured modern JavaScript tutorial.
- [web.dev](https://web.dev/) — Google's guides on performance, PWAs, accessibility, and Core Web Vitals.
