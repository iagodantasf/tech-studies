## Project ideas
- Write end-to-end developer docs for a small open-source CLI: README, API reference, a how-to guide, and a quickstart tutorial — publish with a static docs generator.
- Build a help center for a sample SaaS product: FAQ, troubleshooting guide, and 3 customer-facing how-to articles, structured around real support-request themes.
- Run a mini content-marketing experiment: one pillar page plus 3 supporting blog posts on a technical topic, SEO-optimized with keyword research and tracked for funnel/conversion metrics.
