## Docs & primers
- [Anthropic docs](https://docs.anthropic.com/) — Messages API, tool use, prompt caching, agents.
- [OpenAI docs](https://platform.openai.com/docs) — Responses/Chat API, embeddings, function calling.
- [Hugging Face — LLM course](https://huggingface.co/learn) — transformers, fine-tuning, RAG.
- [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) — spec + SDKs for agent tooling.

## Books / practice
- **Build a Large Language Model (From Scratch)** (Raschka) — how LLMs actually work.
- **AI Engineering** (Chip Huyen) — building applications on foundation models.
- Drill the loop: pick a node → write a `notes/` file → build a tiny thing in `playgrounds/python/` → tick the box.
