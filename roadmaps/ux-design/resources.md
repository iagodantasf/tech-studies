# UX Design — resources

> roadmap.sh: https://roadmap.sh/ux-design

## Books
- **The Design of Everyday Things** (Don Norman) — the foundational text on affordances, signifiers, and human-centered design.
- **Don't Make Me Think** (Steve Krug) — the classic, fast read on web usability and intuitive design.
- **The User Experience Team of One** (Leah Buley) — a practical playbook for doing real UX work with limited resources.
- **About Face** (Alan Cooper) — the definitive deep dive on interaction design principles and patterns.

## Courses / practice
- [Nielsen Norman Group articles](https://www.nngroup.com/articles/) — the gold-standard, evidence-based UX research and usability writing.
- [Google UX Design Certificate (Coursera)](https://www.coursera.org/professional-certificates/google-ux-design) — a structured, beginner-friendly path through the full UX process.
- [Laws of UX](https://lawsofux.com/) — concise reference of the psychological principles every designer should know.
- [Figma Learn](https://help.figma.com/hc/en-us/categories/360002051613-Get-started) — official tutorials for the industry-standard design and prototyping tool.
