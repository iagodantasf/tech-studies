# Forward Deployed Engineer — resources

> roadmap.sh: https://roadmap.sh/forward-deployed-engineer

## Books
- **The Pragmatic Programmer** (David Thomas & Andrew Hunt) — the generalist, pragmatic engineering mindset an FDE lives by.
- **Designing Data-Intensive Applications** (Martin Kleppmann) — the data, storage, and integration foundations behind real customer systems.
- **Crucial Conversations** (Patterson, Grenny, et al.) — handling the high-stakes customer and stakeholder conversations FDEs run constantly.
- **The Mom Test** (Rob Fitzpatrick) — how to do honest discovery and pull real requirements out of customers.

## Courses / practice
- [Palantir — What Is a Forward Deployed Engineer](https://blog.palantir.com/a-day-in-the-life-of-a-palantir-forward-deployed-software-engineer-45ef2de257b6) — the canonical description of the role and how it works in practice.
- [The Twelve-Factor App](https://12factor.net/) — baseline principles for building deployable, portable customer software.
- [Docker — Get Started](https://docs.docker.com/get-started/) — containerization fundamentals for shipping into any environment.
- [Google Cloud Skills Boost](https://www.cloudskillsboost.google/) — hands-on labs to build cloud-deployment fluency.
